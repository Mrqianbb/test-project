# 电商前端项目 - 文件清单

## 项目概述

这是一个基于 Vue 3 + Vite + Element Plus 的电商平台前端项目，与 Spring Cloud 微服务后端完美集成。

## 已创建的文件结构

### 配置文件
- `package.json` - 项目依赖和脚本配置
- `vite.config.js` - Vite 构建工具配置
- `index.html` - HTML 入口文件
- `.gitignore` - Git 忽略文件配置
- `start-frontend.sh` - 前端项目启动脚本（可执行）

### 文档文件
- `README.md` - 项目说明文档
- `QUICK_START.md` - 快速开始指南
- `FRONTEND_ARCHITECTURE.md` - 前端架构详细文档
- `PROJECT_SUMMARY.md` - 本文件（文件清单）

### 源代码文件

#### 入口文件
- `src/main.js` - 应用入口，配置 Vue、路由、状态管理
- `src/App.vue` - 根组件

#### API 接口封装 (src/api/)
- `user.js` - 用户相关接口（注册、登录、获取/更新用户信息）
- `product.js` - 商品相关接口（商品列表、详情、库存更新）
- `order.js` - 订单相关接口（创建、查询、取消订单）
- `payment.js` - 支付相关接口（创建支付、支付详情、完成支付）

#### 公共组件 (src/components/)
- `Header.vue` - 页面头部导航组件（用户信息、购物车入口）
- `Footer.vue` - 页脚组件（链接、联系方式）
- `ProductCard.vue` - 商品卡片组件（商品展示、加入购物车）

#### 路由配置 (src/router/)
- `index.js` - 路由定义和守卫（13个路由，包含登录守卫）

#### 状态管理 (src/stores/)
- `user.js` - 用户状态（token、用户信息、购物车管理）
- `product.js` - 商品状态（商品列表、商品详情）
- `order.js` - 订单状态（订单列表、订单详情、创建/取消订单）

#### 工具函数 (src/utils/)
- `request.js` - Axios 封装（请求/响应拦截器、错误处理）
- `validate.js` - 验证和格式化工具（手机号、邮箱、金额、日期格式化）

#### 页面组件 (src/views/)

##### 首页模块
- `home/Index.vue` - 首页（轮播图、分类展示、热门商品）

##### 用户模块
- `user/Login.vue` - 登录页面（表单验证、登录逻辑）
- `user/Register.vue` - 注册页面（待创建）
- `user/Profile.vue` - 个人中心（待创建）

##### 商品模块
- `product/List.vue` - 商品列表（分类筛选、排序、分页）
- `product/Detail.vue` - 商品详情（待创建）

##### 购物车模块
- `cart/Index.vue` - 购物车（商品列表、数量修改、删除、结算）

##### 订单模块
- `order/Confirm.vue` - 订单确认（收货信息、商品清单、金额计算）
- `order/List.vue` - 订单列表（订单状态、订单详情、取消订单）
- `order/Detail.vue` - 订单详情（待创建）
- `order/Success.vue` - 下单成功（待创建）

##### 支付模块
- `payment/Index.vue` - 支付页面（二维码、支付状态轮询）

### 开发工具

- **unplugin-vue-components** - 自动导入组件
- **Sass** - CSS 预处理器

## 快速开始

### 1. 安装依赖
```bash
cd ecommerce-frontend
npm install
```

### 2. 启动开发服务器
```bash
npm run dev
```

或使用启动脚本：
```bash
bash start-frontend.sh
```

### 3. 访问应用
打开浏览器访问：http://localhost:3000

## 后端服务要求

在启动前端之前，需要确保以下后端服务已启动：

- ✅ Nacos 服务（端口 8848）
- ✅ 网关服务（端口 9000）
- ✅ 用户服务（端口 8001）
- ✅ 商品服务（端口 8002）
- ✅ 订单服务（端口 8003）
- ✅ 支付服务（端口 8004）
- ✅ MySQL 数据库
- ✅ Redis 缓存

## API 接口

所有接口都通过网关服务 `http://localhost:9000` 进行访问。

### 基础路径
- 用户服务：`/users`
- 商品服务：`/products`
- 订单服务：`/orders`
- 支付服务：`/payments`
